Angular + BootStarp Commands
============================
npm install -g @angular/cli --> Angular CLI instllation command.

ng version  --> check Angular CLI installation version.

ng new <project-name> --> create new Angular project.

ng s or [ ng serve ] --> Run the Angular project.

ng g c <component-name> || ng generate component <componet-name> ---> generate component.

ng g s <services-class-name> || ng generate service <services-class-name>  ---> generate new service.

npx kill-port <port Number>  --> kill an existing port which is running.

npm install bootstrap --> install bootstrap into Angular project.

npm install jquery --> install jquery into Angular project.

update your Angular.js file under the scripting [ "bootstreap.min.js", --> relative path ] , styles[ "bootstrap.min.css" --> relative path] of section.

update your styles.css with copied/required css code inorder to use along with BootStarp.

ng init --routing  --> if during new angular project initialization routing moudule is not activate then we can activate it using this command.
